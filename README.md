# Лабораторна робота 7: Система управління студентами та батьками

Цей проєкт реалізує такі функціональні можливості:
- Створення записів для студентів та батьків
- Редагування записів для студентів та батьків
- Перегляд списків студентів та батьків
- Видалення записів для студентів та батьків
